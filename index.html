Built for Performance

Unlike many template engines, Squirrelly doesn't do token parsing. Instead, it uses RegExp, which is backed by C code in JavaScript. Squirrelly consistently outperforms other template engines in benchmarks.

Configurable

Squirrelly supports custom partials, helpers, and filters.

Features

  • Blazing Fast
  • Non-Whitespace Sensitive
  • Not bound to a specific language
  • Custom Helpers
  • Custom Filters
  • Partials
